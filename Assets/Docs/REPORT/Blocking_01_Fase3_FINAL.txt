Fase 3 – Visitor Room (Missione Placeholder Fazione)
BLK-03.01 → Arrivo visitatore (Day 2) + popup “Visitor in attesa” + interazione (E/click) → UI dialogo centro schermo (ritratto placeholder + info fazione + testo con effetto macchina da scrivere)
BLK-03.02 → Missione: scelta percorso (Veloce o Completo) + validazione flow Dome+Lab + logica rifiuto/ritorno (2 giorni).  
  • Quick Path → consegna frutto/pianta già esistente → Ricompensa prevista: pochi CRY + seme nuovo (PLT-102) → Hint: “Soluzione rapida: meno rischi, ma guadagno ridotto.”  
  • Complete Path → coltiva e consegna pianta nuova ottenuta con SDE-001 (≠ starter) → Ricompensa prevista: più CRY + seme nuovo (PLT-102) + 1 item speciale (STR-005 LED Blu o REA-204 Medium A) → Hint: “Soluzione elaborata: più impegno, più valore.”  
  • Se missione RIFIUTATA → il visitatore sparisce e si ripresenta dopo 2 giorni con la stessa richiesta.
BLK-03.03 → Consegna missione: il player deve tornare nella Visitor Room e usare il tasto 📞 Call.  
  • Se missione non completata → auto-reply: “Connessione non disponibile. Torna al lavoro.” (nessun spawn).  
  • Se missione ReadyToDeliver → breve delay con SFX/luce → arrivo visitatore → dialogo finale con riepilogo ricompensa.  
  • Conferma consegna → deduzione item dall’inventario, applicazione ricompensa (in base alla scelta Quick/Complete), log entry nel Diario SPORAE.  
  • Dopo consegna → visitatore lascia la stanza → missione segnata come COMPLETATA.
